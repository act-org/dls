## Simple Alert

You can create a simple alert by just specifying a child description.

<Canvas>
  <Story
    args={{
      children: 'Some alert description goes here.',
    }}
    name="Simple"
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Alert w/ Title

Alerts also support a primary title.

<Canvas>
  <Story
    name="With Title"
    args={{
      children: 'Some alert description goes here.',
      title: 'Title Here',
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

each severity:
